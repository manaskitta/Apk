# 🎨 Figma Clone (Real-time Collaboration)

A minimal Figma-like web application that supports real-time collaboration.  
Users can create designs, draw on canvas, and interact with others live.

## 🚀 Features
- Real-time multi-user collaboration
- Live cursors and interactions
- Drawing shapes and freehand sketches
- Image upload on canvas
- Comments and reactions
- Undo/Redo functionality

## 🛠 Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Fabric.js
- Liveblocks

## ▶️ Run Locally
```bash
git clone <repo-url>
cd <project-folder>
npm install
npm run dev
